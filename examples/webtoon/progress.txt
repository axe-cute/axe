# Webtoon Full-Stack Demo — Build Progress

Goal: Production-ready demo of axe framework. Full stack = Go API + Next.js frontend +
docker-compose all-in-one. UI inspired by Linear design language (dark, minimal, purple accent).

## Decisions
- Design: Linear-inspired (dark, #5E6AD2 purple, Inter font, tight spacing)
- Deploy: local docker-compose only for now
- Scope: Full 5-page Next.js app (Home, Browse, Series detail, Reader, Library, Auth)
